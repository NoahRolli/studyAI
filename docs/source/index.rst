Pallas Documentation
====================

AI-powered study companion that parses lecture notes (PDF, Word, PowerPoint, Excel, images via OCR, Markdown), generates smart summaries, explains key terms, and visualizes knowledge as interactive, zoomable mindmaps. Built with FastAPI, React, and Claude/Ollama. WIP
